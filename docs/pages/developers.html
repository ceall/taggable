Developers

I'm a techy at heart, and so love messing around with other people's code and making it do things I want it to do. If you'd like Taggable to do something unusual, I've exposed the majority of the system to extension hooks, allowing you to manipulate the data at various stages in execution.

taggable_vanilla_tagdata

Called in the class constructor and passed the vanilla, untouched tag data. Useful for parsing custom, module-global template tags.

// taggable_vanilla_tagdata
$this->tagdata = $this->ee->extensions->call('taggable_vanilla_tagdata', $this->tagdata);
if ($this->ee->extensions->end_script === TRUE) return $this->tagdata;

taggable_tags_find_query

Called just before the database find query is executed in {exp:taggable:tags}. Allows you to add custom conditions to the SQL with CodeIgniter's ActiveRecord.

// taggable_tags_find_query
$this->ee->extensions->call('taggable_tags_find_query');

taggable_tags_pre_loop

Called just before the main template loop in {exp:taggable:tags}. Passed the tags and the tagdata, returns the tag array.

// taggable_tags_pre_loop
$tags = $this->ee->extensions->call('taggable_tags_pre_loop', $tags, $this->tagdata);
if ($this->ee->extensions->end_script === TRUE) return $this->tagdata;

taggable_tags_post_loop

Called straight after the main template loop in {exp:taggable:tags}. Passed the template variables to parse and the tagdata, and returns the template variables.

// taggable_tags_post_loop
$vars = $this->ee->extensions->call('taggable_tags_post_loop', $vars, $this->tagdata);
if ($this->ee->extensions->end_script === TRUE) return $this->tagdata;

taggable_tags_end

Called after all finding and parsing has taken place inside {exp:taggable:tags}, and is passed the final tagdata. Returns the tagdata to be outputted.

// taggable_tags_end
$parsed = $this->ee->extensions->call('taggable_tags_end', $parsed);

taggable_ext_tagdata

Called before any processing is made in the extension, passing the tagdata through.

// taggable_ext_tagdata
if ($this->ee->extensions->active_hook('taggable_ext_tagdata')) {
	$tagdata = $this->ee->extensions->call('taggable_ext_tagdata', $tagdata);
	if ($this->ee->extensions->end_script === TRUE) return $tagdata;
}

taggable_ext_chunks

Called after all the tag pair chunks have been found in the extension. Passed the chunks array and tagdata.

// taggable_ext_chunks
if ($this->ee->extensions->active_hook('taggable_ext_chunks')) {
	$tagdata = $this->ee->extensions->call('taggable_ext_chunks', $chunks, $tagdata);
	if ($this->ee->extensions->end_script === TRUE) return $tagdata;
}

taggable_ext_custom_find

Inside the main chunks loop. Allows you to add custom parameters to SELECT tags statement.

// taggable_ext_custom_find
$this->ee->extensions->call('taggable_ext_custom_find', $params, $chunk);

taggable_ext_pre_vars

Called after finding the tags, but before the template vars array is built.

// taggable_ext_pre_vars
$this->ee->extensions->call('taggable_ext_pre_vars', $tags);

taggable_ext_post_vars

Called after the variable array has been built.

// taggable_ext_post_vars
$this->ee->extensions->call('taggable_ext_post_vars', $vars);

taggable_ext_chunk_done

Called after the chunk has been parsed.

// taggable_ext_chunk_done
if ($this->ee->extensions->active_hook('taggable_ext_chunk_done')) {
	$tagdata = $this->ee->extensions->call('taggable_ext_chunk_done', $tagdata);
}

taggable_ext_end

Called after all proccessing has taken place.

// taggable_ext_end
if ($this->ee->extensions->active_hook('taggable_ext_end')) {
	$tagdata = $this->ee->extensions->call('taggable_ext_end', $tagdata);	
}
