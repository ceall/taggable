Introduction

Taggable is an easy-to use, powerful folksonomy engine designed exclusively for ExpressionEngine 2.0. It works with any channel entry, so you can add tags to your blog, news archive, pages, photo gallery - anything thatâ€™s powered by a channel! It features a rich set of template tags that let you list, filter, and display tags and their entries with no code necessary.

About this documentation

This documentation is part reference guide, part tutorial. I'll guide you through the generic process of using Taggable, as well as provide a good thorough reference guide of the tags and their parameters. You're welcome to read the docs through 'cover-to-cover' as it were, as well as use them purely as a reference manual.

Requirements

Taggable only requires an installed copy of ExpressionEngine 2.0.1 and >PHP5.1.

Installation

Install Taggable like any other ExpressionEngine 2 addon. Simply drag the taggable/ folder found in your download into the system/expressionengine/third_party/ folder. Install Taggable like any other ExpressionEngine 2 add-on. Simply drag the third_party/taggable/ folder found in your download into the system/expressionengine/third_party/ folder. Then copy the themes/third_party/taggable/ folder into themes/third_party. Then go into the EE control panel, navigate to the Modules page and find Taggable in the list.

Click the install link and you should see a set of form fields allowing you to enable or disable the various parts of the addon. Make sure you enable every part of the addon - the module and fieldtype.

And you've installed Taggable!

Update

Updating to the latest version of Taggable is really easy. Go to your account page and download the latest version, then replace your current install's folder with the freshly downloaded copy. Then go into the EE control panel, navigate to the Modules page and find Taggable in the list. Click on the Taggable link and Taggable will update itself, then load the dashboard. To confirm that it is indeed updated correctly, go back to the Modules page and check that Taggable's version number is up to date.
